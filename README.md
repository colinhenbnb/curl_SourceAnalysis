#介绍#
* curl全名为"Client for URLs"，即url客户端。
* 由[Daniel Stenberg][1]大神做的开源项目。
* curl由两部分组成
 * curl命令行（curl是利用URL语法在命令行方式下工作的开源文件传输工具）
 * libcurl库（libcurl主要功能就是用不同的协议连接和沟通不同的服务器）
 * （我的理解是：curl是语法工具，libcurl负责底层通信的实现）#待议#
